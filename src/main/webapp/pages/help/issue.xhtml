Choose the “Sprint/Manage Issue” path to come to issue page.

1. Create, update and delete

Click the “New Issue” button and fill in the form information.

You can update the issue by clicking the “edit” button (pencil symbol).

Delete an issue by clicking the “delete” button (bin symbol)..

BACK TO TOP
2. View and history

View detail information and updated history of the issue by clicking the “view” button (magnifier symbol).

BACK TO TOP
3. Excel export

Check the some issues y from table of issues ou want to export to excel file. Choose the file content format (plain text or rich text) and click the “excel” icon..

The file with excel format (.xls extension) will be created.

BACK TO TOP
4. PDF export

Choose the some issues from table of issues you want to export to excel file and click the “pdf” icon.

The file with PDF format will be created.
