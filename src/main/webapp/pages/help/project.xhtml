1. Create project

In “My Project” tab, click the “New Project” button

Fill in the form information and upload your project logo by clicking on “Choose Logo” button

Project Name (*): name of your project.
Description: write something about your project.
Public: a project is public means everyone can see it.
Team: choose the team(s) that will be perform work of the project.

Note: (*) means required field.

Click on “Save” button. The created project will be added to table of your projects

BACK TO TOP
2. View project

Click to project name to go to product backlog page

Select a project from table, click the view detail button

BACK TO TOP
3. Update Project

From the form of project view, click the “Edit Project” button and fill in the information likely to create a project.

Click the “Save” button to finish.
Note: you can not delete any projects instead of updating them.
