1. Navigate to Remaining Task

In “Backlog” tab, choose “Remaining Task”

The second last column will show the new sprint. The third last sprint will show the sprint that the task is not done.

Click "Create New Sprint" to create new sprint if you have not created new sprint yet.

2. Add/Remove the task to new sprint

To add to new sprint check the checkbox then apply. To remove from new sprint uncheck the checkbox then apply
