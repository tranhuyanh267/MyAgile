Choose “Manage Sprint” in menu to go Manage Sprint page

1. Create sprint

After click “New Sprint button”, you have a form below

The system will suggest the start date and the end date of this sprint ( In default, the sprint length is two weeks but you can change this value)

And you can check “Reuse setting from previous sprint” to get information of previous sprint (all fields in dashboard) and use it for the new sprint.
