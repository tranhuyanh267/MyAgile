Click the “My Project/Product Backlog” path to come into product backlog page

1. Create user story

Click the “New” button. Fill in the form information and upload attachment(s).

Name (*): name of the user story.
Priority: accepted values are “Must”, “Should”, “Could” and “Wont”.
Value: value of the user story.
Risk: risk value of the user story.
Description: description of the user story.
ID (-): a number which is used to identify a user story of a project.
Project: the project the user story is belong to.
Status: status of the user story.
Issues: issues belong to the user story.

Note: (*) required field, (-) read only

Click the “Save” button to finish. The created user story will be added to table of your user stories.

Note:

"Showing 2 of 2 Entries": when there are more then 15 user stories, roll down the mouse to see more (lazy loading).

BACK TO TOP
2. Edit user story

From the user stories list, click the Edit icon to show edit form

Fill in the information likely to create a user story

You can set status of the user story to “void” status by clicking the “Set Void”.
Click the “Save” button to finish.

BACK TO TOP

3. Delete user story

From the table of user stories, click the delete button (waste basket symbol). You will get a confirm message and system also send an email noticed to other members of your team

Member can check the email to know what has been happened.

BACK TO TOP
4. Import user story from excel file

Click the “Import Excel File' button

Choose the excel file to upload.

The user stories are created from the excel file will be added to table of user stories
