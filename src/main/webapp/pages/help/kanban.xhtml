1. Key Features:
  • Tasks overview by team and sprint.
  • Task's details.
  • Point Estimation.
  • History of Point.
  • Task Assign.
  • Tasks Relationship.
  • Kanban settings (column colors, column names...).
2. Tasks overview by team and sprint

Tasks are loaded by team and sprint. In order to show tasks of different sprint and team you can do that by clicking on two drop down list and select team or sprint above the kanban

3. Task's detail

Each issue is represented by a yellow sticker. It only shows title of issue when first load. In order to view detail of issue, the user have to double clickon white space of sticker. The detail of issue will be shown.

4. Point Estimation

There are two fields on the bottom left of expanded sticker.

One is Remaining Point (R) and the other is Estimate Point (E). User can update these information by clicking on the text and a text box will appear for inputting new information.

There some notes for working with estimate and remaining point:

  • There are two formats of point DT or plain number. For example: D1T1 or 2
  • User can input point by many ways. For example: D1T1 or 1 1.
  • The remaining point can't be changed while it's in To Do column.
  • The estimate point can only be changed when it's in To Do column.
5. History of Point

User can view history of updated point by clicking on history icon

It will show list of updated history points

6. Task Assign

User can assign task for each member in team by clicking on avatar icon on top right of sticker. An overlay panel will appear for selecting team member.

7. Tasks Relationship

Some tasks which has issue parent will have a relationship icon located on top right and stand next to avatar icon.

When the user clicked on, a pop up will appear and give user a relationship graph between tasks. The red task is the highlight of clicked task.

8. Kanban settings

The settings panel default is hidden and will be shown when user click on "Settings" button.

User can add more column by clicking "Add more Field" button. A text box will appear for user input column name. Note: User can add new column as many as they want.

User can also set color for column by clicking "color rectangle" next to column name.

You can also remove column by clicking "X" button. There are some columns that can't be removed by default: "To do, Done and Accepted".

“Accepted” column is hidden by default, user can show it by clicking “Eye” icon. The difference between “Done” and “Accepted” column is in some situation, the “Done” column means “Done but not yet accepted by PO” and “Accepted” column means “Done and accepted by PO”.
